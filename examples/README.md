# Beispiele

Dieser Ordner enthaelt nur Vorlagen. Die produktiven Dateien liegen nach der
Installation unter `/config`.

## Dateien

- `gerichte.example.json`: oeffentliches Beispiel-Dataset mit genug aktiven
  Gerichten fuer eine komplette Woche.
- `wochenplaene.empty.json`: leerer Startzustand fuer Wochenplaene.
- `configuration-snippet.yaml`: benoetigte Integration, Sensorplattform und
  Shell-Commands.
- `lovelace-resource.yaml`: Ressource fuer die Custom Card.
- `lovelace-views.json`: alle drei Views zusammen.
- `lovelace-view-plan.json`: nur der Wochenplan-View.
- `lovelace-view-new.json`: nur der View fuer neue Gerichte.
- `lovelace-view-edit.json`: nur der Bearbeiten-View.
- `helpers-legacy-snippet.yaml`: optionale Helper fuer eigene Standardkarten
  oder alte Script-Workflows.
- `scripts-legacy-snippet.yaml`: optionale Scripts fuer eigene Buttons oder
  Automationen.

## Datenschutz

Die Beispieldaten sind neutral und enthalten keine privaten Originalgerichte.
