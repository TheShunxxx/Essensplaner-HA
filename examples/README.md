# Beispiele

Dieser Ordner enthält nur Vorlagen. Die produktiven Dateien liegen nach der
Installation unter `/config`.

## Dateien

- `gerichte.example.json`: öffentliches Beispiel-Dataset mit genug aktiven
  Gerichten für eine komplette Woche.
- `wochenplaene.empty.json`: leerer Startzustand für Wochenpläne.
- `configuration-snippet.yaml`: optionales YAML-Setup, falls die Integration
  nicht über die UI hinzugefügt werden soll.
- `lovelace-resource.yaml`: Ressource für die Custom Card.
- `lovelace-views.json`: alle drei Views zusammen.
- `lovelace-view-plan.json`: nur der Wochenplan-View.
- `lovelace-view-new.json`: nur der View für neue Gerichte.
- `lovelace-view-edit.json`: nur der Bearbeiten-View.

## Datenschutz

Die Beispieldaten sind neutral und enthalten keine privaten Originalgerichte.
